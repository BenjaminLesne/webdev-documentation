## Others
### Merge request
```markdown
# Toutes les cases doivent être cochées avant la fusion.

- [ ] Le commit suit la convention : [type]:[ticket number] [description], 1ère ligne <= 50 cacaractères.
- [ ] Le code est expliqué dans la description de la merge request
- [ ] Les travaux temporaires sont supprimés (code commenté, console logs, etc).
- [ ] Des tests ont été ajoutés ou mis à jours lorsque nécessaire.
- [ ] Une vidéo montrant que le code fonctionne avec la console navigateur affichée est ajoutée à la section “Vidéos”([logiciel pour enregistrer](https://obsproject.com/))

# De quoi s’agit-il ?

%{all_commits}

# Videos
```
